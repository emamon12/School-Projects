Developed on Netbeans IDE 8.2
Java version 8 update 191 (build 1.8.0_191-b12)

To build just type make in the respective directories. In Windows, type make outside the directory, but inside the respective source folders.

To run the server type: server1.Server <port-number>

To run the client type: client.Client <host-name> <port-number>

This must be done in their respective folders.
