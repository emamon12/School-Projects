Developed on Visual Studio Code 1.37.1
Java version 8 update 211 (build 1.8.0_211-b12)

For Windows: To compile, type "make" in the SequenceAlignment directory. To run, type "run" in the directory. To delete class files, type "clean".

For Linux/Mac: To compile, type "make" in the SequenceAlignment directory. To run, type "make run" in the directory. To delete class files, type "make clean".

Put input files inside the input folder within the SequenceAlignment directory.

For inputs, all white spaces are removed and all lines, after the the first line, are concatenated.

The code is formatted as closely as possible to the perl code provided.

For code issues please email me at emamon@siue.edu.
